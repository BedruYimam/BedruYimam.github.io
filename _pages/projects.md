# Projects & Software

## Software Projects

### 1. **EthioNLP Toolkit**
**Status:** Active Development | **Tech:** Python, Hugging Face

Open-source natural language processing tools for Ethiopian languages.

**Features:**
- Amharic text preprocessing
- Pretrained language models
- Named entity recognition
- Sentiment analysis

**GitHub:** [github.com/BedruYimam/EthioNLP](https://github.com/BedruYimam/EthioNLP)

### 2. **Ethiopian Student Analytics Platform**
**Status:** Pilot Phase | **Tech:** Django, React, PostgreSQL

Dashboard for tracking and analyzing student performance in Ethiopian universities.

**Components:**
- Student performance prediction
- Early warning system
- Visualization tools
- Reporting module

### 3. **Amharic OCR System**
**Status:** Research Prototype | **Tech:** TensorFlow, OpenCV

Optical character recognition system for Amharic documents.
